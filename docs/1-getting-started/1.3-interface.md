# 1.3 界面概览

RPG Agent MV 的主界面由编辑器、Agent 面板和控制台组成。三者分别负责工程视图、自然语言任务和项目管理。

## 编辑器

编辑器用于查看地图、处理事件和管理待放置事件。它不是 RPG Maker MV 原生编辑器的完整替代。

主要能力包括：

- 在地图树中选择地图，并在画布中查看对应地图。
- 在地图模式中查看地图，使用已接入的受控图块、阴影和区域编辑能力。
- 在事件模式中处理事件的新建、编辑、移动、复制、粘贴和删除。
- 放置 Agent 生成的待放置事件。
- 应用或丢弃暂存改动。
- 使用顶部全局 **Play / Stop** 启动或停止当前工程根目录的 `Game.exe`。互动试玩运行源工程，不包含暂存内容；工程可能写入自己的 `save/`。

如果工程没有可加载地图，编辑器会显示空态提示。此时应先检查项目接入状态。

## Agent 面板

Agent 面板用于提交自然语言需求并查看任务结果。

主要能力包括：

- 输入事件生成、批量修改、脚本或插件相关需求。
- 回答 ASK 确认。
- 查看 Agent 输出、事件预览和运行结果。
- 在运行过程中停止当前任务。

## 控制台

控制台用于查看项目状态和管理配置。

主要入口包括：

- **资产库**：查看地图、技能、图块、角色、图片和音频，并识别明显引用问题。
- **项目管理**：查看和编辑 15 类数据库与公共事件；数据库和公共事件草稿可撤销/重做，保存到暂存后历史会清空。
- **插件**：通过原生文件选择器安装 `.js`；新装默认禁用，覆盖保留原配置，删除时文件和配置一起进入暂存。
- **素材**：按分类导入，查看引用数量，并在可安全改写全部已知引用时改名；仅零引用素材可以删除。
- **运行日志**：查看 Agent 执行记录、项目检查结果和引擎输出。
- **设置**：管理模型配置、工具策略和项目偏好。
