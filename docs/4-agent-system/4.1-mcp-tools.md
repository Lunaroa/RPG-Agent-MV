# 4.1 MCP 工具

## 定义

MCP 工具是 Agent 访问外部能力的接口。RPG-Agent-MV-MV 通过 MCP 工具读取 RMMV 项目事实，并执行地图、事件等受控操作。工具调用记录会出现在控制台运行日志中。

## 内置 RMMV 工具

RPG-Agent-MV-MV 内置一组 RMMV 专用 MCP 工具，覆盖读取上下文、地图操作和事件操作。

| 工具 | 作用 |
|------|------|
| **读上下文**（`RmmvReadContext`） | 读取事件上下文、地图索引、素材清点、插件清点、数据库目录、开关变量状态等项目事实 |
| **地图**（`RmmvMap`） | 执行地图创建、属性修改、复制、删除、绘制，以及暂存应用或丢弃 |
| **事件**（`RmmvEvent`） | 新事件：`registry.register` → 右侧预览 → 用户「应用到待放置队列」→ 桌面端地图拖放（`createPlacementEvent`）。已有事件：补丁、反馈、`editor.update`/`editor.move`（**禁止**用于新事件落点或引导用户去外部 RMMV 编辑器） |

这些工具是 Agent 在 RMMV 工程中工作的核心能力，默认启用。

## 调用场景

- 写入事件前，Agent 调用读上下文工具获取项目事实。
- 涉及地图修改或事件落地时，Agent 调用地图和事件工具。
- 调用结果写入运行日志，供用户核对任务执行过程。

## 扩展 MCP 工具

如需接入产品内置能力之外的 MCP 服务器，需在高级配置中注册目标服务器、授权对应工具，并通过简单任务确认可用性。外部 MCP 属于扩展能力，其稳定性取决于对应服务器本身。
