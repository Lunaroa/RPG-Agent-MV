# 5.4 MZ 1.10.0 官方兼容证据

本页记录 RPG Agent MV 当前支持范围内的“官方要求 → 实现位置 → 定向测试”。规格证据只采用 RPG Maker 官方网站、官方帮助和官方脚本参考。用户工程只用于本机临时副本验收，不会进入文档、测试、fixture、日志或 Git。

## 固定范围

- 接受数据和核心脚本完整、核心版本可识别的 MZ 工程目录；不把 `game.rmmzproject` 当作导入前提。MZ 1.10.0 是完整验证基线；旧版核心和加密资源需确认兼容警告。
- 地图采用官方 **自动层 + 第 1～4 层**，另保留格式自带的阴影层和区域层；不实现无限图层或私有地图格式。
- 不复刻角色生成器、地牢生成器、DLC、教程、部署工具或 MZ 编辑器本体。
- 编辑不要求安装 MZ。试玩、战斗测试和粒子预览使用工程自带且经过静态校验的 NW.js 运行环境。

## 官方资料不一致时

官方通用数据库帮助页仍保留较旧的 2000/1000 说明，而官方产品功能页明确列出角色、职业、状态、技能、物品、武器、防具、敌人、敌群和公共事件可设为 9999。本产品只对这十类采用 9999，不把该上限外推到动画或图块集。

如果官方网页、PDF 与 1.10.0 核心实际格式出现其他冲突，对应写入必须停止并增加可复现测试；不能靠猜测、降级结构或自动改写核心继续。

## 产品自有安全约束

以下属于 RPG Agent MV 自己的安全规则，不冒充 MZ 官方规格：所有改动先进入暂存；新事件落点由用户确认；历史 MV 会话和工具名保持兼容；私有工程不得进入测试、fixture、文档、日志或 Git。
